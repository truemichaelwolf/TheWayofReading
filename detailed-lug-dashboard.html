Fundamental Principles | 基本原则

Chunking | 分块

Breaking down discourse into intuitive chunks

将话语分解成直观的语块

"Speech needs to be processed fast, or it is lost forever"
"语言需要快速处理，否则就会永远丢失"

Linearity | 线性

Processing language in real-time sequence

实时顺序处理语言

Message Elements (M-Types) | 信息元素

Type | 类型 Description | 描述 Example | 示例
M (Basic) Complete, independent units | 完整独立单元 "birds don't see it and pick it off"
MS (Supplement) Additional details | 补充细节 "in order to be cryptic"
MR (Revision) Reformulation | 重新表述 "in order for it to be hidden"

Organization Elements (O-Types) | 组织元素

Type | 类型 Function | 功能 Examples | 示例
OI (Interactive) Managing interaction | 管理互动 okay, well, oh dear
OT (Textual) Organizing discourse | 组织话语 and so on, for example

Analysis Process | 分析过程

  1. Chunking | 分块
    • Identify natural boundaries | 识别自然边界
    • Average chunk length: 3-9 words | 平均语块长度：3-9个词
  2. Element Classification | 元素分类
    • Message (M) vs Organization (O) | 信息与组织
    • Subtypes identification | 子类型识别

Chunking Patterns | 分块模式

Processing Evidence | 处理证据

  • Real-time processing constraints | 实时处理限制
