Method Overview

The multivariate collexeme analysis method is designed to identify distinctive patterns across multiple dimensions that differentiate between two seemingly similar constructions:

GO (a)round and V

Examples:

  • "They're going around and paying off all of these countries."
  • "He's going around and spying on us."

GO (a)round Ving

Examples:

  • "Well I don't go around assaulting people."
  • "You shouldn't go around fighting strangers."

Methodology Goals

The research aims to:

  • Extend collostructional analysis beyond individual lexemes
  • Include multiple contextual dimensions simultaneously
  • Identify complex patterns of feature interactions
  • Create comprehensive "collo-profiles" of constructions
  • Demonstrate how context contributes to constructional knowledge

Corpus Selection

The study used a 248,145,425-word portion of the Corpus of Contemporary American English (COCA) covering the time period 2010-2019. All COCA registers were represented: academic, fiction, magazines, movies, news, spoken, and TV.

Four search strings were used to retrieve instances of the constructions:

Association Rules Method

The core of the multivariate collexeme analysis is based on association rules, a data mining technique that identifies notable co-occurrences of levels of different variables.

Conceptual Explanation

Association rules work like recommendation systems: "if a person buys product X, they are likely to also buy product Y." Each rule consists of:

  • Left-hand side (LHS): A set of variable levels that form the condition
  • Right-hand side (RHS): A variable level that forms the consequence
If [features in LHS are present], then [feature in RHS is likely to be present]

Key Statistics

Support
Proportion of cases with both LHS and RHS
Confidence
Proportion of cases with LHS that also have RHS
Lift
Ratio of observed support to expected support

Example Rule

Rule Format:

If colligation is modalized VP and modality is deontic and polarity is affirmative, then construction is GO (a)round and V

Component Value
LHS colligation = modalized VP, modality = deontic, polarity = affirmative
RHS construction = GO (a)round and V
Frequency 6
Support 0.011
Confidence 0.545
Lift 2.938

Parameter Settings

  • Minimum Support: 0.01 (rule must apply to at least 1% of data)
  • Minimum Confidence: 0.5 (rule must be correct at least 50% of the time)
  • Focus: Only rules with construction in RHS were retained
  • Redundancy Elimination: Rules with more general versions were removed

Software and Algorithm

# Implementation in R with arules package library(arules) # Set parameters parameter_settings <- list( support = 0.01, confidence = 0.5 ) # Apply apriori algorithm rules <- apriori(data, parameter = parameter_settings) # Filter rules with constructions in RHS construction_rules <- subset(rules, subset = rhs %in% "construction=GO (a)round and V" | rhs %in% "construction=GO (a)round Ving") # Remove redundant rules non_redundant_rules <- is.redundant(construction_rules) pruned_rules <- construction_rules[!non_redundant_rules] # Sort by lift sorted_rules <- sort(pruned_rules, by="lift")

Advantages of Multivariate Approach

Comparison with Traditional Analysis

A traditional distinctive collexeme analysis would have yielded only minimal information:

Traditional Analysis Results

GO (a)round Ving: Only identifies "kill, do, pretend" as distinctive collexemes

Multivariate Analysis Advantage

Reveals complex patterns such as:

  • GO (a)round and V associates with neutral discourse prosody in spoken register
  • GO (a)round Ving associates with negative discourse prosody and unpleasant semantic prosodies in fiction/TV/movies

Theoretical Implications

  • Supports viewing constructions as multidimensional entities
  • Reinforces distinction between internal and external constructional properties
  • Aligns with usage-based tenet that context is crucial to language systems
  • Demonstrates that constructional differences extend beyond lexical preferences
  • Provides empirical basis for constructional "collo-profiles"
"It makes sense to assume that many constructions are essentially multidimensional entities, and application of multivariate methods, such as association rules, is thus a useful, if not even necessary, way of identifying how external properties interact and, in the case of comparing two or more constructions, the extent to which such interactions are distinctive."

Methodological Contributions

  • Introduces a new approach to collostructional analysis
  • Demonstrates how to incorporate multiple contextual dimensions
  • Provides framework for analyzing complex feature interactions
  • Shows effectiveness even with relatively small datasets
  • Can be extended to other constructional comparisons

方法概述

多变量搭配词分析方法旨在识别多维度上的区别模式，以区分两种表面上相似的构式：

GO (a)round and V

例子：

  • "They're going around and paying off all of these countries."（他们到处付钱给这些国家）
  • "He's going around and spying on us."（他到处监视我们）

GO (a)round Ving

例子：

  • "Well I don't go around assaulting people."（我不会到处攻击人）
  • "You shouldn't go around fighting strangers."（你不应该到处与陌生人打架）

方法目标

该研究旨在：

  • 将搭配结构分析扩展到单个词汇之外
  • 同时包含多个语境维度
  • 识别特征相互作用的复杂模式
  • 创建构式的综合"搭配概况"
  • 展示语境如何贡献于构式知识

关联规则方法

多变量搭配词分析的核心基于关联规则，这是一种数据挖掘技术，用于识别不同变量级别之间的显著共现。

概念解释

关联规则的工作原理类似于推荐系统："如果一个人购买产品X，他们也可能购买产品Y"。每条规则包括：

  • 左侧(LHS)：形成条件的一组变量级别
  • 右侧(RHS)：形成结果的变量级别
如果[存在LHS中的特征]，那么[RHS中的特征很可能存在]

关键统计数据

支持度
同时具有LHS和RHS的案例比例
置信度
具有LHS的案例中也具有RHS的比例
提升度
观察到的支持度与预期支持度的比率

规则示例

规则格式：

如果词类搭配是情态化动词短语且情态是道义情态且极性是肯定的，那么构式是GO (a)round and V

组成部分 值
左侧(LHS) 词类搭配 = 情态化动词短语，情态 = 道义情态，极性 = 肯定
右侧(RHS) 构式 = GO (a)round and V
频率 6
支持度 0.011
置信度 0.545
提升度 2.938

参数设置

  • 最小支持度： 0.01（规则必须适用于至少1%的数据）
  • 最小置信度： 0.5（规则必须至少在50%的时间内正确）
  • 焦点： 仅保留右侧包含构式的规则
  • 冗余消除： 删除有更一般版本的规则

多变量方法的优势

与传统分析的比较

传统的区别性搭配词分析只能提供最少的信息：

传统分析结果

GO (a)round and V：仅识别"introduce, check, round"作为区别性搭配词

GO (a)round Ving：仅识别"kill, do, pretend"作为区别性搭配词

多变量分析优势

揭示了复杂模式，例如：

  • GO (a)round and V与口语语域中的中性话语韵律相关联
  • GO (a)round Ving与小说/电视/电影中的负面话语韵律和不愉快的语义韵律相关联

理论意义

  • 支持将构式视为多维实体
  • 强化了构式内部属性和外部属性之间的区别
  • 符合基于使用的原则，即语境对语言系统至关重要
  • 证明构式差异超越了词汇偏好
  • 为构式"搭配概况"提供了实证基础
"假设许多构式本质上是多维实体是有意义的，因此，应用多变量方法（如关联规则）是识别外部属性如何相互作用的有用甚至必要的方式，在比较两个或多个构式的情况下，这种相互作用在多大程度上具有区别性。"

方法论贡献

  • 引入了搭配结构分析的新方法
  • 展示了如何整合多个语境维度
  • 提供了分析复杂特征相互作用的框架
  • 显示了即使在相对较小的数据集上也能有效
  • 可以扩展到其他构式比较
