Construction Network Classifications / 构式网络分类系统

Goldberg's Classification (1995) / Goldberg的分类

"Early on, Goldberg (1995: 74-81) proposed the following four types of constructional links."
Page 224

1. Instance Links / 实例连接

Links between constructions of different degrees of specificity
不同特定程度构式之间的连接

2. Polysemy Links / 多义连接

Links between prototypical sense and its extensions
原型意义与其扩展意义之间的连接

3. Metaphorical Extension Links / 隐喻扩展连接

Links between basic sense and metaphorically extended sense
基本意义与隐喻扩展意义之间的连接

4. Subpart Links / 部分连接

Links between constructions of different degrees of complexity
不同复杂程度构式之间的连接

Diessel's Classification (2019) / Diessel的分类

"Diessel (2019: 12f., 22) lists the following six relations."
Page 224

1. Symbolic Relations / 符号关系

Associations between form and meaning
形式与意义之间的联系

2. Sequential Relations / 序列关系

Associations between linguistic elements in sequence
序列中语言元素之间的联系

3. Taxonomic Relations / 分类关系

Associations between representations at different levels of specificity
不同特定程度表征之间的联系

4. Lexical Relations / 词汇关系

Associations between lexemes
词位之间的联系

5. Constructional Relations / 构式关系

Associations between constructions
构式之间的联系

6. Filler-slot Relations / 填充槽位关系

Associations between particular items and slots of constructions
特定项目与构式槽位之间的联系

Vertical Links / 垂直连接

"Constructions are vertically related via so-called 'inheritance' links."
Page 224

Complete Inheritance Model / 完全继承模型

Information is stored non-redundantly at the highest possible level
信息非冗余地存储在最高可能层级

Default Inheritance Model / 默认继承模型

Only non-conflicting information is inherited from above
只继承上层的非冲突信息

Full Entry Model / 完整条目模型

All information is specified redundantly in every node
所有信息在每个节点都冗余指定

Horizontal Links / 水平连接

"The 'constructicon' is not a top-down radiating diagram but, rather, a lattice."
Page 229

1. Differential Opposition Links / 差异对立连接

Links based on minimal formal differences
基于最小形式差异的连接

2. Allostruction Links / 异构式连接

Links between alternating constructions expressing near-synonymous meanings
表达近义意义的交替构式之间的连接

3. Analogical Links / 类比连接

Links based on formal similarity influencing each other
基于形式相似性相互影响的连接

4. Polysemy/Metaphorical Links / 多义/隐喻连接

Links between semantically related meanings
语义相关意义之间的连接

5. Syntagmatic Links / 组合关系连接

Links between elements that co-occur in sequence
序列中共现元素之间的连接
